# jewelstack

## 📱 Overview

**JewelStack** is a comprehensive mobile application designed specifically for jewelry shop owners in Karnataka, India. It addresses critical challenges of inventory management, gold price volatility, alloy composition accuracy, and order tracking.

### 🎯 Problem Statement

Small jewelry shop owners face multiple challenges:
- 📦 Manual inventory tracking leading to stock discrepancies
- ⚖️ Alloy calculation errors causing BIS hallmark failure
- 📈 Gold price volatility affecting profit margins
- 📋 Order management chaos with no customer updates
- 👥 No customer purchase history tracking

### ✨ Solution

JewelStack provides a complete digital solution with:
- ✅ Dual user roles (Owner & Staff with different access levels)
- ✅ Smart inventory management with real-time tracking
- ✅ Alloy & composition calculator for accurate pricing
- ✅ Live gold rate integration with automatic updates
- ✅ Order progress tracking with daily timeline
- ✅ Complete customer profiles with purchase history
- ✅ Professional invoice generation (PDF & DOC)
- ✅ ML-powered diamond price prediction

## 🚀 Key Features

### 👑 Owner Access (Full Control)
| Feature | Description |
|---------|-------------|
| Inventory Management | Add, edit, delete jewelry items |
| Alloy Calculator | 22K/18K/24K purity calculations |
| Price Management | Update gold rates, set making charges |
| Order Control | Create, modify, cancel orders |
| Progress Timeline | Update order status daily |
| Invoice Generation | PDF & DOC with business branding |
| ML Predictor | AI-powered diamond pricing |
| Staff Management | Create staff accounts, set permissions |

### 👥 Staff Access (Limited Access)
| Feature | Description |
|---------|-------------|
| View Inventory | See stock availability |
| Create Orders | With pre-set prices |
| Update Progress | Add daily order updates |
| View Customers | Access customer details |
| Generate Invoices | Pre-approved formats |
